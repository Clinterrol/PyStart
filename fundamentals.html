Python Fundamentals

The basic building blocks every Python beginner needs to know

1 Variables & Data Types

Variables store information that your program can use later. Python automatically detects what type of data you are storing — no need to declare the type manually unlike Java or C.

name = "Real" # String
age = 20 # Integer
gpa = 1.75 # Float
is_student = True # Boolean
print(name, age, gpa, is_student)

2 Conditional Statements

Conditionals let your program make decisions. Python uses indentation to define blocks — no curly braces needed.

grade = 88

if grade >= 90:
print("Excellent!")
elif grade >= 75:
print("Passed")
else:
print("Try again")

3 Loops

Loops repeat a block of code multiple times. The for loop runs a fixed number of times, while the while loop runs as long as a condition is true.

# For loop
for i in range(5):
print("Count:", i)

# While loop
x = 0
while x < 3:
print("x is", x)
x += 1

4 Functions

Functions are reusable blocks of code. Define them once and call them whenever needed — avoiding repetition and making your code cleaner.

def greet(name):
return "Hello, " + name + "!"

def add(a, b):
return a + b

print(greet("Real"))
print(add(10, 5))

5 Lists

Lists store multiple values in a single variable. You can access, add, remove, and loop through items easily — one of the most used data structures in Python.

subjects = ["Python", "Web Dev", "InfoSec"]

print(subjects[0]) # Python
subjects.append("Java")
print(len(subjects)) # 4

for s in subjects:
print(s)
