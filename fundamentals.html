Python Fundamentals

The basic building blocks every Python beginner needs to know

Variables & Data Types

Variables store information that your program can use later. Python automatically detects what type of data you are storing — you don't need to declare the type manually.

name = "Real"      # String
age = 20         # Integer
gpa = 1.75       # Float
is_student = True # Boolean
print(name, age, gpa, is_student)

Conditional Statements

Conditionals let your program make decisions. If a condition is true, one block of code runs. Otherwise, a different block runs.

grade = 88

if grade >= 90:
  print("Excellent!")
elif grade >= 75:
  print("Passed")
else:
  print("Try again")

Loops

Loops repeat a block of code multiple times. The for loop runs a fixed number of times, while the while loop runs as long as a condition is true.

# For loop
for i in range(5):
  print("Count:", i)

# While loop
x = 0
while x < 3:
  print("x is", x)
  x += 1

Functions

Functions are reusable blocks of code. You define them once and call them whenever you need them — avoiding repetition and making your code cleaner.

def greet(name):
  return "Hello, " + name + "!"

def add(a, b):
  return a + b

print(greet("Real"))
print(add(10, 5))

Lists

Lists store multiple values in a single variable. You can access, add, remove, and loop through items in a list easily.

subjects = ["Python", "Web Dev", "InfoSec"]

print(subjects[0])    # Python
subjects.append("Java")
print(len(subjects))  # 4

for s in subjects:
  print(s)
