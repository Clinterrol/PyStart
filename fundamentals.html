Python Fundamentals

Master the core building blocks of Python with clean, interactive code blueprints.

1. Variables & Data Types

Variables act as storage containers for your program data. Python automatically detects the data type without explicit declaration.

# Dynamic variable assignment
user_name = "Alice" # String
user_age = 21 # Integer
gpa = 3.75 # Float
is_learning = True # Boolean

print(user_name, "is", user_age)
Output: Alice is 21
2. Conditionals (if-elif-else)

Conditionals control the flow of execution based on true/false conditions using clean indentation.

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
else:
    print("Grade: C")
Output: Grade: B
3. Loops (For & While)

Loops iterate repeatedly over sequences or check boolean statuses to automate tasks efficiently.

# Looping through a sequence range
for i in range(3):
    print("Iteration:", i)

# Quick Python list array iteration
fruits = ["apple", "banana"]
for f in fruits:
    print(f)
Output:
Iteration: 0
Iteration: 1
Iteration: 2
apple
banana
4. Functions

Functions collect reusable code snippets into named logic sequences to prevent repetitive structures.

def calculate_area(width, height):
    return width * height

result = calculate_area(5, 4)
print("Area calculation:", result)
Output: Area calculation: 20
