Python Origins

The History of Python

From a holiday project to the world's most popular programming language

i Who Created Python?

Python was created by Guido van Rossum, a Dutch programmer who wanted to build a language that was easy to read, simple to write, and fun to use. He was working at Centrum Wiskunde & Informatica (CWI) in the Netherlands when he started the project. Guido served as Python's "Benevolent Dictator For Life" (BDFL) for decades before stepping down in 2018.

Timeline

Key milestones in Python's evolution
1989
The Beginning

During the Christmas holiday of 1989, Guido van Rossum began writing the first version of Python as a side project. He wanted a language that improved upon ABC while fixing its shortcomings. The name "Python" was inspired by the British comedy group Monty Python's Flying Circus.

1991
Python 1.0 Released

Python 1.0 was officially released in February 1991. It already included exception handling, functions, and core data types like lists, dictionaries, and strings. It was open source from the very beginning, which helped it grow a community quickly.

2000
Python 2.0 Released

Python 2.0 introduced major new features including list comprehensions, garbage collection, and Unicode support. This version was widely adopted for many years. However, Python 2 officially reached its end of life on January 1, 2020 and is no longer supported.

2008
Python 3.0 Released

Python 3.0 was a major revision intentionally not backward compatible with Python 2. It fixed fundamental design flaws, cleaned up inconsistencies, and modernized the language. Today, Python 3 is the only supported version and is what all beginners should learn.

Now
Python Today

Python consistently ranks as the #1 most popular programming language in the world. It is used by companies like Google, Instagram, Spotify, NASA, and Netflix. Its simplicity, versatility, and massive community make it the top choice for beginners and experts alike.
