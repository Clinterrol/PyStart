Installation Guide

Easy, step-by-step documentation on setting up Python and VS Code environment.

Environment Setup

Pumili ng operating system sa ibaba upang makita ang tamang proseso ng pag-install.

Windows Installation Steps

  1. I-download ang Python: Pumunta sa opisyal na Python Website at i-download ang pinakabagong installer para sa Windows.
  2. Napakahalaga: Siguraduhing lagyan ng tsek (check) ang kahon na may nakasulat na "Add Python.exe to PATH" bago i-click ang Install Now.
  3. I-verify ang Setup: Buksan ang inyong Command Prompt (CMD) at i-type ang code na ito upang suriin kung pumasok ang system environment:
    python --version
  4. I-setup ang Editor: I-download at buksan ang Visual Studio Code, pagkatapos ay i-install ang "Python Extension" na gawa ng Microsoft.

macOS Installation Steps

  1. Gumamit ng Homebrew (Inirerekomenda): Buksan ang Terminal app ng Mac at patakbuhin ang code na ito para sa mabilis na package management package:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. I-install ang Python 3: Kapag may Homebrew na ang Mac ninyo, patakbuhin lang ang simpleng command na ito:
    brew install python
  3. I-verify ang Setup: Tiyakin kung gumagana sa pamamagitan ng pag-type nito sa terminal terminal app:
    python3 --version
