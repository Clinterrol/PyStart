About PyIntro

PyIntro is a student-built web project created as the Final Integrative Activity for Web Development Fundamentals at National Teachers College. The goal of this site is to introduce Python Programming to complete beginners — covering its history, purpose, real-world uses, and basic code fundamentals in a clean, accessible format.

The Course

This project was built using Bootstrap 5 for responsive design, Git and GitHub for version control and deployment, and PHP for server-side form handling. It represents the culmination of three weeks of learning across Bootstrap components, Git workflows, and PHP back-end basics.
