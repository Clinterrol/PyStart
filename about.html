About Our Team

Meet the creators behind PyIntro — Web Development Fundamentals Project.

The PyIntro Project

Ang PyIntro ay isang integrative web platform na binuo para sa aming kulminatibong aktibidad sa Web Development Fundamentals. Ang layunin ng proyektong ito ay magbigay ng malinaw, madaling intindihin, at interactive na panimula sa wikang Python para sa mga nagsisimula pa lamang mag-code.

Pinagsasama ng site na ito ang responsive design ng Bootstrap 5, dynamic front-end form interceptors gamit ang JavaScript, at server-side logic gamit ang PHP para sa aming data capture form systems.

Built for Academic Year 2025–2026
